# SurveyTask

## Запуск проекта
Для запуска проекта на вашем компьютере вам понадобится:

- Склонируйте репозиторий: git clone https://github.com/IWorldManI/SurveyTask.git
- Перейдите в директорию проекта
- Запустите проект с помощью Docker Compose: ```docker-compose up -d ```
- Обратите внимание на каком порту запущено приложение используя: ```docker-compose ps```
- API будет доступно по адресу: http://localhost:(PORT)/swagger/index.html
- Скрипт для тестовых данных [`SQL_Script`](./SQL_Script.txt)
